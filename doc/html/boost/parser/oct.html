Global oct

boost::parser::oct

Synopsis

// In header: <boost/parser/parser.hpp>

constexpr parser_interface< uint_parser< unsigned int, 8 > > oct;

Description

The octal unsigned integer parser. Produces an unsigned int attribute. To parse a particular value x, use oct(x).
