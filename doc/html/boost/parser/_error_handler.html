Function template _error_handler

boost::parser::_error_handler

Synopsis

// In header: <boost/parser/parser.hpp>


template<typename Context> 
  decltype(auto) _error_handler(Context const & context);

Description

Returns a reference to the error handler object associated with the innermost parser. Returns none if there is no associated error handler.
