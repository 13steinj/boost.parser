Type definition integral_constant

integral_constant

Synopsis

// In header: <boost/parser/tuple.hpp>


typedef hana::integral_constant< T, I > integral_constant;

Description

A template alias that is std::integral_constant<T, I> when either BOOST_PARSER_STANDALONE or BOOST_PARSER_USE_STD_TUPLE is defined, and boost::hana::integral_constant<T, I> otherwise.
