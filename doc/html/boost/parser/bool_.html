Global bool_

boost::parser::bool_

Synopsis

// In header: <boost/parser/parser.hpp>

constexpr parser_interface< bool_parser > bool_;

Description

The Boolean parser. Parses "true" and "false", producing attributes true and false, respectively, and fails on any other input.
